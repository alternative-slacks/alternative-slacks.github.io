About

alt-bot is a Slack bot that provides information about questionable news sources. It searches over 400 fake news sources based on user input. Once alt-bot receives a successful query, it returns information about the fake news source's title, URL, category (satire, clickbait, conspiracy, etc.) and political alignment.

alt-bot uses data from the Fake News Detector GitHub repository and uses botpress, an open-source platform for building bots.

Created by alternative-slacks, a Chingu Walrus team.
