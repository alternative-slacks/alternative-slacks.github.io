About

alt-slack is a Slack bot that provides information about questionable news sources. It searches over 400 fake news sources based on user input. Once the bot receives a successful query, it returns information about the fake news source's title, URL, category (satire, clickbait, conspiracy, etc.) and political alignment.

The alt-slack bot uses data from the Fake News Detector GitHub repository and uses the botpress platform.

Created by alternative-slacks, a Chingu Walrus cohort.
